# Invoice Generator — 請求書自動作成ツール

## 概要
フォームに情報を入力するだけで、プロフェッショナルな請求書をリアルタイムプレビュー付きで作成し、PDF出力できるWebアプリケーションです。

## なぜ作ったか
請求書をExcelやWordで毎回作るのは手間がかかります。ブラウザだけで簡単に作成・PDF保存できるツールがあれば便利だと考え、開発しました。

## 機能一覧
- 基本情報の入力（請求書番号・請求日・支払期限）
- 請求先・発行者情報の入力
- 品目の追加・削除（何行でも追加可能）
- 数量 × 単価の自動計算
- 消費税の自動計算（税率変更可能）
- リアルタイムプレビュー（入力と同時に右側で確認）
- PDF出力 / 印刷機能
- 全入力クリア機能
- レスポンシブ対応

## 使用技術
| 種類 | 技術 |
|------|------|
| フロントエンド | HTML / CSS / JavaScript |
| レイアウト | CSS Grid |
| PDF出力 | ブラウザ印刷機能（window.print） |

## 技術的なポイント
- 単一HTMLファイルで完結（サーバー不要）
- リアルタイムプレビューをイベントリスナーで実装
- 品目行の動的な追加・削除を配列管理で実現
- 印刷時にフォーム部分を非表示にするCSS @media print の活用
- XSS対策としてユーザー入力をエスケープ処理

## 使い方
1. `invoice.html` をブラウザで開く
2. 基本情報・請求先・自社情報を入力
3. 品目を入力（行の追加・削除可能）
4. 右側のプレビューで確認
5. 「PDF出力」ボタンでPDF保存・印刷
