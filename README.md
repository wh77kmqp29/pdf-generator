# PDF Generator（PDF自動作成ツール）

## 概要
手動で情報を入力すると、自動でPDFを生成するWebアプリです。

## なぜ作ったか
書類作成は手作業だと時間がかかり、ミスも起きやすい。
入力フォームに必要事項を入れるだけで
整ったPDFが自動生成される仕組みを作りたかった。

## 機能一覧
- 入力フォーム（名前・日付・内容など）
- 入力内容のプレビュー表示
- PDF生成＆ダウンロード
- テンプレート切り替え

## 使用技術（予定）
| 種類 | 技術 |
|------|------|
| フロントエンド | HTML / CSS / JavaScript |
| バックエンド | Python (Flask) |
| PDF生成 | ReportLab または WeasyPrint |
